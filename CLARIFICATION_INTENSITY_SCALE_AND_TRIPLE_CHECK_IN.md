# 🎯 Esclarecimento: Intensidade 1-5 + Ritual Duplo (Manhã/Tarde/Noite)

## 📊 O que significa Intensidade 1-5?

### Escala de Intensidade de Energia

```
INTENSIDADE = "Quanto dessa energia você está depositando AGORA?"

1/5 - MÍNIMA (Apenas tocando)
├─ Foco Profundo 1/5: Lendo código, sem criar nada
├─ Criativo 1/5: Pensando devagar em ideias
├─ Físico 1/5: Caminhada leve, alongamento
└─ Exemplo: "Estou aqui, mas não dou 100%"

2/5 - BAIXA (Presente, mas cansado)
├─ Foco Profundo 2/5: Debugando, concentração mediana
├─ Criativo 2/5: Juntando ideias existentes
├─ Social 2/5: Conversando com alguém, mas sem energia
└─ Exemplo: "Estou funcionando, mas cansado"

3/5 - MODERADA (Normal, equilibrado)
├─ Foco Profundo 3/5: Escrevendo features, concentrado
├─ Criativo 3/5: Ideando coisas novas, fluxo ok
├─ Colaborativo 3/5: Pair programming normal
└─ Exemplo: "Estou bem, fazendo meu trabalho normalmente"

4/5 - ALTA (Fluxo!)
├─ Foco Profundo 4/5: Deep work intenso, na zona
├─ Criativo 4/5: Ideias fluem, brainstorm produtivo
├─ Físico 4/5: Exercício intenso, suando
└─ Exemplo: "Estou no flow! Consigo fazer tudo!"

5/5 - MÁXIMA (Pico!)
├─ Foco Profundo 5/5: Programação complexa, hiperfoco
├─ Criativo 5/5: Brainstorm explosivo, ideias demais
├─ Restaurador 5/5: Meditação profunda, totalmente descansado
└─ Exemplo: "Estou no meu melhor! Nada me para!"
```

### Diferença Prática

```
Sua tarefa: Codificar feature X (requer Foco Profundo)

Se registra: Foco Profundo 1/5
└─ "Conseguirei fazer, mas será lento e com bugs"
└─ Ash recomenda: "Não é o melhor timing. Deixa para quando estiver 3/5+"

Se registra: Foco Profundo 3/5
└─ "Consigo fazer normalmente"
└─ Ash recomenda: "Bom momento para começar"

Se registra: Foco Profundo 5/5
└─ "Vou resolver tudo hoje!"
└─ Ash recomenda: "AGORA! Use esse pico!"
```

### Por Que Importa?

```
Ash precisa entender:
├─ Você tem energia? 5/5 = sim, máxima
├─ Você está cansado? 1/5 = sim, muito
├─ Você pode fazer a tarefa agora? 3/5+ = sim
└─ Quando voltar a este nível? (Padrão detectado)

EXEMPLO SEMANAL:
Segunda:  Criativo 5/5 → Execute projeto novo
Terça:    Foco 4/5 → Implemente ideias
Quarta:   Admin 2/5 → Só email, leve
Quinta:   Criativo 3/5 → Planejamento
Sexta:    Restaurador 4/5 → Descanso recuperador

Ash aprende: "Você é criativo de segunda. Quarta cansa. Sexta se recupera bem."
```

## 🔄 Ritual Duplo/Triplo: Manhã + Tarde + Noite

### Por Que 3 Momentos?

```
PROBLEMA ANTERIOR:
└─ 1 check-in por dia = Perde mudanças ao longo do dia

SOLUÇÃO:
├─ MANHÃ: Check-in de energia inicial (como acordou)
├─ TARDE: Check-in de energia (mudou durante o dia?)
└─ NOITE: Estado emocional (reflexão do dia)

Ash consegue ver:
"Segunda você acordou Criativo 5/5
 À tarde virou Foco Profundo 4/5
 À noite se sentiu Entusiasmado → Confiante
 
 Padrão: Você começa criativo, implementa com foco, termina confiante!"
```

## ✅ Resumo Final

### Os 3 Check-ins Diários

```
MANHÃ (06:00-08:00)
├─ Ash pergunta proativamente
├─ Energy Type principal
├─ Intensidade 1-5
├─ Opcional: Teve transição durante a manhã?
└─ Tempo: 30s

TARDE (14:00-16:00)
├─ Ash pergunta proativamente
├─ Energy Type (mudou vs manhã?)
├─ Intensidade 1-5
├─ Compara com manhã automaticamente
└─ Tempo: 30s

NOITE (19:00-21:00)
├─ User abre diário (reflexivo)
├─ Escreve sobre o dia
├─ Seleciona até 3 estados emocionais
├─ Ash conecta tudo: energias + emoções + contexto

```
1 CHECK-IN (Antigo):
└─ "Você foi criativo" (genérico)

├─ Manhã: Criativo 5/5
├─ Tarde: Foco 4/5 (mudou!)
└─ Noite: Entusiasmado → Confiante → Grato (contexto)
   └─ Ash: "Você teve PADRÃO CRIAÇÃO PURA perfeito!
            Comece assim toda segunda-feira."
