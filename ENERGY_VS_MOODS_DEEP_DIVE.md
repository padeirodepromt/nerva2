## 🧬 A Diferença Conceitual

### **ENERGY TYPES - "Tipos de Energia" (O QUE VOCÊ CONSEGUE FAZER)**

```
Definição: Categorias que descrevem A QUALIDADE E O TIPO DE TRABALHO 
           que você consegue fazer bem em um determinado momento.

Tempo de Variação: ⏱️ Horas a dias
Estabilidade: 📊 Relativamente estável (muda com circadiano e ciclo)
Preditor de: 🎯 PERFORMANCE e PRODUTIVIDADE
Exemplos:

┌─────────────────────────────────────────────────────────────────────┐
│ FOCO PROFUNDO (⚡ Indigo)                                           │
│ Quando: Manhã, após dormir bem                                     │
│ Características: Imersão total, deep work, sem distrações          │
│ Trabalhos ideais: Programação, análise, escrita, design complexo  │
│ Duração típica: 4-6 horas contínuas                               │
│ Ritmo: Lento, deliberado, perfeccionista                          │
├─────────────────────────────────────────────────────────────────────┤
│ CRIATIVO (🎨 Pink)                                                  │
│ Quando: Tranquilo, sem pressão, após inspiração                   │
│ Características: Ideação, inovação, expressão livre               │
│ Trabalhos ideais: Brainstorm, arte, copy, inovação               │
│ Duração típica: 2-4 horas                                         │
│ Ritmo: Fluido, exploratório, experimental                        │
├─────────────────────────────────────────────────────────────────────┤
│ ADMIN (📋 Slate)                                                    │
│ Quando: Qualquer hora, não requer pico de energia                │
│ Características: Rotina, operacional, organização                 │
│ Trabalhos ideais: Email, planilhas, documentação, arquivos       │
│ Duração típica: 1-2 horas em sprints                             │
│ Ritmo: Automático, mecânico, eficiente                           │
├─────────────────────────────────────────────────────────────────────┤
│ CONEXÃO (❤️ Emerald)                                               │
│ Quando: Após conexão social, com energia social                  │
│ Características: Relacionamentos, colaboração, empatia           │
│ Trabalhos ideais: Reuniões 1:1, mentoria, feedback, apoio       │
│ Duração típica: 2-3 horas                                        │
│ Ritmo: Empático, atento, presente                                │
├─────────────────────────────────────────────────────────────────────┤
│ RESTAURADOR (🌿 Teal)                                              │
│ Quando: Quando energizado está baixo, necessita recuperação      │
│ Características: Descanso, autocuidado, recuperação              │
│ Trabalhos ideais: Yoga, meditação, passeio, leitura leve        │
│ Duração típica: 1-3 horas                                        │
│ Ritmo: Lento, confortável, acolhedor                             │
├─────────────────────────────────────────────────────────────────────┤
│ SOCIAL (👥 Blue)                                                    │
│ Quando: Com energia, pós-cafeína, humor social alto              │
│ Características: Comunidade, grupo, evento, vibração coletiva   │
│ Trabalhos ideais: Networking, eventos, palestras, workshops    │
│ Duração típica: 3-5 horas                                        │
│ Ritmo: Dinâmico, energizado, conectado                           │
├─────────────────────────────────────────────────────────────────────┤
│ REFLEXIVO (🧘 Purple)                                              │
│ Quando: Tranquilo, solitude, introspecção necessária            │
│ Características: Meditação, journaling, análise profunda        │
│ Trabalhos ideais: Diário, planejamento pessoal, meditação      │
│ Duração típica: 1-2 horas                                       │
│ Ritmo: Lento, introspectivo, contemplativo                      │
├─────────────────────────────────────────────────────────────────────┤
│ FÍSICO (💪 Amber)                                                   │
│ Quando: Corpo quer se mexer, energia vitalizada                 │
│ Características: Movimento, exercício, corpo, vitalidade        │
│ Trabalhos ideais: Academia, caminhada, dança, yoga dinâmico    │
│ Duração típica: 1-2 horas                                       │
│ Ritmo: Dinâmico, vitalizado, presente                           │
├─────────────────────────────────────────────────────────────────────┤
│ ESTRATÉGICO (🎯 Red)                                               │
│ Quando: Bem dormido, alerta mental, sem stress agudo            │
│ Características: Planejamento, análise, decisão, visão          │
│ Trabalhos ideais: Planejamento trimestral, decisões, roadmap  │
│ Duração típica: 2-3 horas                                       │
│ Ritmo: Analítico, deliberado, pensante                          │
```

### **MOODS - "Estados Emocionais" (COMO VOCÊ SE SENTE)**

```
Definição: Emoções instantâneas que descrevem COMO VOCÊ SE SENTE 
           neste momento específico.

Tempo de Variação: ⏱️ Minutos a poucas horas
Estabilidade: 🌊 Muito fluida (responde a eventos imediatos)
Preditor de: 💭 BEM-ESTAR e SAÚDE MENTAL
Exemplos:

┌─────────────────────────────────────────────────────────────────────┐
│ CALMO (😌 Purple)                                                   │
│ Triggers: Meditação, chá quente, ambiente silencioso              │
│ Sensação: Paz, equilíbrio, sem pressa                             │
│ Duração: 15min - 2h                                               │
│ Próximo estado possível: Alegre → Reflexivo                      │
├─────────────────────────────────────────────────────────────────────┤
│ ALEGRE (😊 Yellow)                                                  │
│ Triggers: Boa notícia, música, conexão social, café               │
│ Sensação: Otimismo, esperança, leveza                            │
│ Duração: 30min - 4h                                              │
│ Próximo estado possível: Criativo → Ansioso                     │
├─────────────────────────────────────────────────────────────────────┤
│ FOCADO (🎯 Blue)                                                    │
│ Triggers: Deadline, tarefa importante, música, cafeína           │
│ Sensação: Concentração, determinação, intencionalidade          │
│ Duração: 1h - 4h                                                 │
│ Próximo estado possível: Criativo → Ansioso                     │
├─────────────────────────────────────────────────────────────────────┤
│ CRIATIVO (✨ Pink)                                                  │
│ Triggers: Inspiração, conversa, tema interessante, autonomia    │
│ Sensação: Inspiração, fluxo, aliveness                           │
│ Duração: 30min - 3h                                              │
│ Próximo estado possível: Focado → Confuso                       │
├─────────────────────────────────────────────────────────────────────┤
│ ANSIOSO (😰 Red)                                                    │
│ Triggers: Deadline urgente, evento importante, notícia ruim      │
│ Sensação: Preocupação, tensão, adrenalina                        │
│ Duração: 15min - 8h                                              │
│ Próximo estado possível: Focado → Triste                        │
├─────────────────────────────────────────────────────────────────────┤
│ CONFUSO (😕 Slate)                                                 │
│ Triggers: Ambiguidade, múltiplas opções, falta de informação   │
│ Sensação: Incerteza, desorientação, indecisão                   │
│ Duração: 30min - 2h                                              │
│ Próximo estado possível: Reflexivo → Focado                     │
├─────────────────────────────────────────────────────────────────────┤
│ GRATO (🙏 Green)                                                    │
│ Triggers: Reflexão, sucesso, conexão significativa, presente     │
│ Sensação: Reconhecimento, apreciação, plenitude                 │
│ Duração: 1h - 6h                                                 │
│ Próximo estado possível: Calmo → Alegre                         │
├─────────────────────────────────────────────────────────────────────┤
│ TRISTE (😢 Blue)                                                    │
│ Triggers: Perda, fracasso, isolamento, decepção                 │
│ Sensação: Luto, tristeza, introversão                            │
│ Duração: 1h - vários dias                                        │
│ Próximo estado possível: Reflexivo → Calmo                      │
└─────────────────────────────────────────────────────────────────────┘
```

### **Exemplos Práticos**

**Cenário 1: Perfeito Alinhamento**
```
Energy: Criativo (🎨)
Mood: Criativo (✨)
Resultado: ●● FLOW STATE - Momento ideal para criar
Indicador Ash: "Seu momentum criativo está máximo. 
               Capture essa energia em um projeto!"
```

**Cenário 2: Desalinhamento Útil**
```
Energy: Estratégico (🎯)
Mood: Ansioso (😰)
Resultado: ◑ PRODUTIVO COM TENSÃO
Indicador Ash: "Sua ansiedade está amplificando 
               seu planejamento. Use isso para 
               clarificar problemas complexos."
```

**Cenário 3: Desalinhamento Problemático**
```
Energy: Foco Profundo (⚡)
Mood: Triste (😢)
Resultado: ○ IMPRODUTIVO
Indicador Ash: "Você está em isolamento emocional.
               Considere conectar com alguém 
               antes de trabalho profundo."
```

## 🎯 Decisão de Design: Como Usar Ambos

### **Para o Check-in**

O check-in deve perguntaR DUAS COISAS DIFERENTES:

```javascript
// Check-in Form
1. "Que tipo de energia você tem?" 
   → Seleciona UM dos 9 ENERGY_TAGS
   → Questão: O QUE VOCÊ CONSEGUE FAZER?
   
2. "Como você se sente emocionalmente?"
   → Seleciona UM dos 8 MOODS
   → Questão: COMO VOCÊ SE SENTE?
   
3. "Em uma escala, qual sua energia geral?" (1-5)
   → Numérico
   → Questão: INTENSIDADE DE ENERGIA
```

### **Para Analytics**

```
Dashboard mostra:
┌─ EnergyStatsCard
│  └─ Distribuição de tipos de energia (últimos 7 dias)
│  └─ Padrão: "Você tem 40% foco, 25% criativo, 35% admin"
│
├─ MoodStatsCard
│  └─ Distribuição de moods (últimos 7 dias)
│  └─ Padrão: "Você está 60% alegre/grato, 15% ansioso"
│
└─ HolisticInsightsCard
   └─ CORRELAÇÕES: "Quando você está criativo + alegre,
                    completa 80% mais tarefas. Mantenha
                    essa combinação!"
```

### **Para Recomendações**

```
Ash pode fazer recomendações baseadas em AMBOS:

IF energyType = "restaurador" THEN
  Suggest: Yoga, meditação, passeio (task type)
  
IF mood = "ansioso" THEN
  Suggest: Respiração 4-7-8, meditação guiada (ritual)
  
IF energyType = "foco_profundo" AND mood = "criativo" THEN
  Suggest: Trabalhe em projeto desafiador AGORA
           (sinergia máxima detectada)
  
IF energyType = "estrategico" AND mood = "confuso" THEN
  Suggest: Clarify (reflective practice first)
           antes de decidir
```

## 🎓 Conclusão

**States Emocionais** (MOODS) e **Tipos de Energia** (ENERGY_TAGS) são **dimensões complementares**:

- Use **ENERGY_TAGS** para: *"O que você consegue fazer agora?"*
- Use **MOODS** para: *"Como você está se sentindo agora?"*
- Combine ambos para: *"Qual é a melhor ação para você agora?"*

Isso transforma o Prana de um **gestor de tarefas** para um **assistente de vida**.
