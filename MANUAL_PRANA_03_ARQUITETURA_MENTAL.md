## 🧬 CONCEITOS FUNDAMENTAIS

### O Que é Prana, Neurologicamente?

Prana é um **sistema operacional neural** baseado em três pilares:

```
┌─────────────────────────────────────┐
│        PRANA NEURAL SYSTEM          │
├─────────────────────────────────────┤
│  1. INPUTS                          │
│     └─ Energy Check-ins             │
│     └─ Mood Registrations           │
│     └─ Task Creations               │
│     └─ Journal Entries              │
│                                     │
│  2. PROCESSING (SINAPSE)            │
│     └─ Energy Analysis Service      │
│     └─ Holistic Analysis            │
│     └─ AI Agent (Ash)               │
│     └─ Astrological Insights        │
│                                     │
│  3. OUTPUTS                         │
│     └─ Dashboard Insights           │
│     └─ Recommendations              │
│     └─ Sankalpa (Daily Intention)   │
│     └─ Task Intelligence            │
│     └─ Memory Archive (Papyrus)     │
└─────────────────────────────────────┘
```

### Os Neurônios de Prana

Cada "neurônio" é um **componente funcional** que processa tipos específicos de dados:

#### **Neurônio 1: Energy Neuron**
- **Função:** Rastreia e analisa energia do usuário
- **Inputs:** Energy check-ins (1-10 escala)
- **Processing:** Análise histórica de 7 dias
- **Outputs:** Energy stats, trends, recomendações
- **Como funciona:** Quando você registra seu nível de energia, o sistema armazena essa informação e analisa a tendência dos últimos 7 dias. Com essa análise, recomenda quais tarefas são mais adequadas ao seu estado energético atual

#### **Neurônio 2: Mood Neuron**
- **Função:** Mapeia estados emocionais
- **Inputs:** Mood selections (Happy, Anxious, Calm, Energized, etc)
- **Processing:** Correlação com energy, tarefas, eventos
- **Outputs:** Mood distribution, emotional patterns
- **Como funciona:** Registra seu estado emocional e correlaciona com tarefas e eventos. Identifica padrões: que atividades causam ansiedade, quais trazem paz, como seu humor varia ao longo do mês

#### **Neurônio 3: Spark Neuron**
- **Função:** Captura e armazena ideias rápidas
- **Inputs:** Quick ideas (voice, text, sketch)
- **Processing:** Categorização automática
- **Outputs:** Inbox de ideias, projetos sugeridos
- **Como funciona:** Você expressa uma ideia qualquer (texto, fala, desenho). O sistema captura, categoriza automaticamente e sugere se deveria virar um projeto novo. Evita que ideias se percam

#### **Neurônio 4: Project Neuron**
- **Função:** Organiza trabalho em contextos maiores
- **Inputs:** Project creation, task grouping
- **Processing:** Hierarchia, relacionamentos, views
- **Outputs:** Múltiplas visualizações (Sheet, Kanban, Mindmap)
- **Como funciona:** Agrupa tarefas relacionadas em projetos. Cada projeto pode ser visto em múltiplas perspectivas: Sheet (tabular), Kanban (fluxo de status), Mindmap (relações visuais). Mesmos dados, visualizações diferentes

#### **Neurônio 5: Task Neuron**
- **Função:** Unidade atômica de ação
- **Inputs:** Task creation com metadata (priority, energy, time)
- **Processing:** Scheduling, energy matching, prioritização
- **Outputs:** Task lists, recommendations, smart scheduling
- **Como funciona:** Cada tarefa tem metadata: quanto de energia requer, quanto tempo leva, qual prioridade. Sistema usa isso para sugerir melhor momento para fazer, baseado em seu padrão de energia

#### **Neurônio 6: Diary Neuron**
- **Função:** Memória viva do usuário (Papyrus)
- **Inputs:** Journal entries, sketches, notes
- **Processing:** Indexação, search, memory enhancement
- **Outputs:** Insights retrospectivos, memory archive
- **Como funciona:** Você escreve reflexões no Diário. O sistema indexa, permite buscar e conecta com padrões passados. Papyrus é sua memória inteligente que encontra padrões nas suas anotações

#### **Neurônio 7: Astro Neuron**
- **Função:** Integra dados astrológicos
- **Inputs:** Birth chart, current date/time
- **Processing:** Ephemeris calculations, transits
- **Outputs:** Astral insights, daily sankalpa, recommendations
- **Como funciona:** Usa seu mapa natal e os transitos atuais para gerar insights astrológicos. Combina com sua energia/humor para criar recomendações personalizadas (sankalpa - intenção do dia)

#### **Neurônio 8: AI Neuron (Ash)**
- **Função:** Inteligência artificial unificada
- **Inputs:** Todos os outros neurônios (dados contextuais)
- **Processing:** Language model (GPT-4.1), prompt engineering
- **Outputs:** Recomendações inteligentes, escritura automática
- **Como funciona:** Ash recebe contexto de todos os neurônios (sua energia, humor, tarefas recentes, anotações, dados astrológicos) e gera recomendações personalizadas. É o maestro que orquestra toda a inteligência

## 🔌 AS SINAPSES: COMO OS NEURÔNIOS SE CONECTAM

Uma "sinapse" em Prana é uma **conexão de dados** entre neurônios:

### Sinapse 1: Energy → Task Recommendation

Quando você registra alto nível de energia, o sistema automaticamente recomenda tarefas que combinam com esse estado energético. Tarefas que exigem muita energia são priorizadas, enquanto tarefas mais leves ficam para outro momento.

### Sinapse 2: Mood + Diary → Insights

Seu estado emocional registrado é conectado com o que você escreve no Diário. O sistema identifica padrões: "Você fica ansioso quando tem prazos" ou "Você fica calmo depois de completar projetos". Ash usa esses padrões para gerar insights personalizados.

### Sinapse 3: Energy + Astral → Sankalpa

Seu nível de energia atual é combinado com dados astrológicos do dia (transitos, posições) e seu estado de humor. Isso gera uma intenção personalizada para o dia (sankalpa): uma frase que resume o melhor para você fazer naquele dia específico.

### Sinapse 4: Task → Diary Reflection

Quando você completa uma tarefa, o sistema sugere que você reflita sobre ela no Diário. Essa reflexão é conectada com a tarefa no sistema, criando um conhecimento permanente: o que você aprendeu, como se sentiu, que desafios encontrou.

## 🔄 FLUXO DE DADOS COMPLETO (EXEMPLO REAL)

Digamos que um usuário está usando Prana em um dia típico:

### **08:00 - Morning Check-in**
```
Usuário faz Energy Check-in (7/10) + Mood (Calm, Focused)
    ↓
Energy Neuron: Registra check-in, calcula 7-day trend (↗ subindo)
Mood Neuron: Registra humor do dia
    ↓
Trigger: Sinapses ativadas
    ├─ Energy + Mood → Sankalpa Engine dispara
    ├─ Energy Level + Tasks → Smart sorting
    └─ Astral Data + Energy → Astrological insight
    ↓
Output para Dashboard:
  - Sankalpa: "Aproveite seu estado de foco para criar"
  - Energy Stats: Trend positivo (gráfico)
  - Mood Stats: Distribuição de humores da semana
  - Task Recommendations: Tarefas de alta energia listadas
  - Astrology Card: Transito favorável para criatividade
```

### **14:00 - Working + Spark Capture**
```
Usuário cria 3 Sparks (ideias rápidas)
    ↓
Spark Neuron: Captura, auto-categoriza
    ↓
Ash anota: Todas ideias estão em tema "Produto"
    ↓
AI Neuron (Ash) sugere:
  "Seus Sparks parecem indicar novo projeto: UI Redesign"
    ↓
Usuário clica "Create Project"
    ↓
Project Neuron: Cria projeto base, conecta Sparks como tasks
    ↓
Task Neuron: Gera 5 tasks iniciais com base nos sparks
    ↓
Output: Novo projeto "UI Redesign" com 5 tasks prontas
```

### **18:00 - End of Day Reflection**
```
Usuário escreve Diary Entry
    ↓
Diary Neuron: Índica entrada
    ↓
Papyrus Search: Encontra entradas similares no passado
    ↓
Ash Analysis: Extrai 3 key insights da entrada
    ↓
Holistic Service: Conecta entrada com tarefas do dia
    ↓
Output no Diary:
  - "Você mencionou isso antes em [data X]"
  - "Você completou 5 tarefas hoje (82% do planejado)"
  - "Seu mood hoje (Satisfied) contrasta com [semana passada]"
  - "Para próxima vez: evitar X quando estiver com Y energia"
```

### **21:00 - Sleep Analysis**
```
Usuário faz Evening Check-in (5/10, Relaxed)
    ↓
Energy Neuron: Registra queda normal de energia
Mood Neuron: Registra estado relaxado
    ↓
Holistic Analysis:
  - Correlação: Completed tasks → Relaxed mood ✓
  - Padrão: Energia cai entre 20:00-21:00 (normal circadiano)
    ↓
Next Day Sankalpa Prep:
  - Ash: Analisa dia para gerar sankalpa de amanhã
  - Astro: Calcula transitos para amanhã à 08:00
    ↓
Sleep recommendation (via Papyrus):
  - "Você dormiu bem quando completou tarefas"
  - "Amanhã tem Lua Minguante: bom dia para finalizar"
```
