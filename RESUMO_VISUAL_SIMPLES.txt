EXEMPLO 3: Chat Manual (Sem Componente)
───────────────────────────────────────

  import { useOlly } from '@/contexts/OllyContext'

  export default function MinhaPage() {
    const { chat, messages, isLoading } = useOlly()

    return (
      <div>
        {messages.map(msg => <p key={msg.id}>{msg.content}</p>)}
        <button onClick={() => chat('Olá!')}>Enviar</button>
      </div>
    )
  }

SE ALGO NÃO FUNCIONAR:
═══════════════════════════════════════════════════════════════════════════

  ❌ "DATABASE_URL não definido"
     → Verifique .env se tem a variável

  ❌ "useOlly() é undefined"
     → Certifique-se que envolveu App com <OllyProvider>

  ❌ "Tabelas não encontradas"
     → Execute: node setup-olly-db.js

  ❌ "Falha ao conectar ao Olly"
     → Verifique se .env tem VITE_OLLY_API_URL
