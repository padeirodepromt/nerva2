## 🆘 Emergency Numbers

Se tudo der ruim:

```
Chat não responde?
→ Check: OPENAI_API_KEY ou GEMINI_API_KEY
→ Check: Backend em /api/ai/chat
→ Logs: console.log em chatService.js

Tool não funciona?
→ Check: Exportada em toolService.js
→ Check: Tem declaration e handler
→ Teste: curl -X POST /api/ai/chat

View não renderiza?
→ Check: Importada em ToolCallResultCard
→ Check: Switch case correto
→ Check: Suspense fallback mostra?

Dança não funciona?
→ Check: window.addEventListener setup
→ Check: Links formato [Texto](task:id)
→ Check: CSS .highlight-pulse definido
```
